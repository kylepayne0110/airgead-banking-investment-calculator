\# Airgead Banking Investment Calculator



\## Overview

This project is a C++ console application developed for the SNHU CS-210 Programming Languages course.



The application allows users to explore how investments grow over time using compound interest.



Users can enter:

\- Initial investment amount

\- Monthly deposit

\- Annual interest rate

\- Number of years



The program then displays two reports:

1\. Growth without monthly deposits

2\. Growth with monthly deposits



---



\## Features

\- Object-oriented design using C++

\- Input validation for all user inputs

\- Monthly compound interest calculations

\- Clear, formatted console output



---



\## Technologies Used

\- C++

\- Standard Library (iostream, iomanip, string)
